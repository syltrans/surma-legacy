Syloti-Nagri keyboard for Keyman

This keyboard layout is designed for typing in the Syloti Nagri script, using the Surma L font, available from http://www.sylheti.org.uk. It includes an on screen keyboard which can be viewed by clicking the Keyman icon and selecting the On Screen Keyboard menu item.
